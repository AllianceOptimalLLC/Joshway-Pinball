# Joshway Pinball

**Courageous hero multi-table pinball adventure themed around Joshway.**

Play classic pinball with flippers, ramps, multi-ball, power-ups, and high scores. **5 unique tables** inspired by the Joshway worlds:

1. Adventure Island (easy jungle ramps)
2. Courage City (skyscraper loops)
3. Star Fortress (fast tight expert)
4. **Joshway's Hideout** (cozy living-room furniture, grand loop, expert)
5. **Cosmic Realm** (NEW - wormholes, blackhole arcs, comet boosts, nebula secrets)

Includes **Hero Challenge Mode** (full rings = time-limited max multiplier frenzy up to x9.5+), tilt/nudge, arcs/loops, 6 power-ups incl Cape + Shield + Cosmic, rich procedural music per table.

## Features
- **5 unique Joshway-themed tables** (Adventure Island, Courage City, Star Fortress, **Joshway's Hideout**, **NEW Cosmic Realm** — wormholes, blackhole secrets, generated cosmic assets)
- Realistic 2D physics: flippers, bumpers, slingshots, **arc loops**, ramps with boosts
- Multi-ball, score multipliers (up to x10.5), 6 power-ups (multiball, multiplier, extraball, **capeboost**, **shield**, **cosmicboost** ☄️ using image-gen asset)
- Tilt scoring bonus + bumper combo chain (x1-12) for extra points on consecutive hits. Hero mode x2.4 + detailed end bonuses.
- Secrets on Cosmic Realm (blackhole loops trigger powers/bonuses)
- **Hero Challenge Mode** on full ring clear (time-limited high-score frenzy)
- Chiptune music (table-specific melodies) + rich procedural SFX (Web Audio, thumps, chimes)
- Tilt/nudge (Arrow keys), classic pinball feel
- Local high scores + leaderboards per table (persistent top 5)
- Retro pixel-perfect UI with CRT scanlines, vignette, floating particles, hero branding
- Responsive (scales on smaller screens)

## Controls
- Z / Shift / Left-Right arrows or mouse — Flippers (left/right)
- Space / click — Plunger launch
- Arrow keys — Tilt / Nudge (classic pinball)
- P — Pause, M — Mute music, R — Retry level
- Full keyboard + touch friendly
