# Joshway Pinball

**Sonic-style multi-table pinball adventure themed around Joshway.**

Play classic pinball with flippers, ramps, multi-ball, power-ups, and high scores. Multiple unique tables inspired by the Joshway worlds (Living Room, Starfield, Mars, etc.).

## Live Demo
- **Production**: https://joshway-pinball.vercel.app
- Vercel project: joshway-pinball (AllianceOptimalLLC)

## Features
- Multiple tables with Joshway hero theme
- Realistic 2D physics: flippers, bumpers, slingshots, loops
- Multi-ball, score multipliers, power-ups
- Chiptune music + rich SFX (Web Audio)
- Local high scores + leaderboards per table
- Retro pixel-perfect UI matching the Joshway series

## Controls
- Mouse or keyboard to control flippers (left/right)
- Space / click for plunger launch
- Nudge table carefully!

## Run Locally
```bash
npm install
npm run dev
# or build: npm run build
```
